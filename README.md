# Virtual-Library
Books (Library)

Функциональный сайт для хранения книг с возможностью добавления, просмотра и оценки.

**Фокус:**
* Работа с шаблонами (Jinja2)
* Использование Bootstrap для верстки
* Минимизация дублирования кода
